BODY

Do português CORPO, é usado para identificar o corpo do texto.


É o elemento do HTML que representa o conteúdo de um documento HTML. Imagine que você está construindo um aplicativo de cadastro de usuários. Quando um novo usuário preenche o formulário de registro e envia os dados, uma requisição é feita ao servidor. Os dados do usuário, como nome, email e senha, são incluídos no "body" dessa requisição.
